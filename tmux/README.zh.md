# tmux 配置

Fork 自 [gpakosz/.tmux](https://github.com/gpakosz/.tmux)。自定义了状态栏、快捷键和默认行为。

## 要点

- **前缀键**：`C-b`（默认）
- **历史**：5000 行
- **索引**：窗口和面板从 1 开始
- **鼠标**：默认启用，`C-b m` 切换
- **256色**：`screen-256color` 终端类型
- **焦点事件**：开启

## 自定义快捷键

### 面板管理
| 快捷键 | 动作 |
|--------|------|
| `C-b s` | 垂直分割 |
| `C-b v` | 水平分割 |
| `C-b x` | 关闭面板 |
| `C-b +` | 最大化/恢复面板 |
| `C-b h/j/k/l` | 选择面板（vim 风格） |
| `C-b H/J/K/L` | 调整面板大小（2 格） |

### 窗口管理
| 快捷键 | 动作 |
|--------|------|
| `C-b C-h` | 上一个窗口 |
| `C-b C-l` | 下一个窗口 |
| `C-b Tab` | 上一个窗口 |

### 其他
| 快捷键 | 动作 |
|--------|------|
| `C-b r` | 重新加载配置 |
| `C-b m` | 切换鼠标 |
| `C-b Enter` | 进入复制模式 |
| `C-b b` | 列出缓冲区 |
| `C-b p` | 粘贴缓冲区 |
| `C-b P` | 选择缓冲区 |
