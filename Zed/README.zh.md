# Zed 编辑器配置说明

本目录包含 Zed 编辑器的配置文件。

## 配置文件

- `settings.json` — Zed 主配置文件

## 配置要点

- **Vim 模式**：启用，含相对行号、智能大小写搜索
- **剪贴板**：始终使用系统剪贴板
- **字体**：UI 16px，编辑器 15px
- **Tab 宽度**：2
- **自动保存**：焦点改变时
- **主题**：跟随系统（One Light / One Dark）
- **光标闪烁**：启用
- **Yank 高亮**：200ms 持续时间

## Vim 模式功能

### 标准操作
- `h/j/k/l` — 移动光标
- `g d` — 跳转到定义
- `g D` — 跳转到声明
- `g y` — 跳转到类型定义
- `g I` — 跳转到实现
- `c d` — 重命名
- `g A` — 查找所有引用
- `g s` — 在文件中查找符号
- `g S` — 在项目中查找符号

查看所有可用设置：命令面板运行 `zed: open default settings`。
