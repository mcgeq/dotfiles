A production-ready Vim configuration written in *Vim9script* with hybrid modular architecture.

---

* Overview

This configuration uses a *hybrid architecture* that combines:
- =modules/= for complex feature modules with config dicts and health checks
- =config/= for simple settings and language-specific configurations

** Architecture Layers

#+BEGIN_SRC text
Bootstrap → Core → Modules → Config → Local
    ↓         ↓        ↓         ↓        ↓
 Environ   Utils   Features  Simple   User
#+END_SRC

- *Bootstrap* - Environment initialization, constants, basic settings
- *Core* - Error handling, utilities, module loader, health check
- *Modules* - Feature modules (20 files)
- *Config* - Simple configurations (8 files)
- *Local* - User customizations

** Performance

- Startup time: ~80-100ms
- Smart lazy loading
- Built-in health checks

* Installation

** Requirements

| Requirement     | Version | Purpose           |
|-----------------+---------+-------------------|
| Vim/Neovim      | 9.0+/0.8+ | Editor            |
| Node.js         | 16+     | CoC.nvim          |
| Git             | any     | Version control   |
| ripgrep (rg)    | any     | Fast search       |
| fd              | any     | File finding      |
| ctags           | any     | Code tags         |

** Quick Start

*** Windows

#+BEGIN_SRC powershell
winget install BurntSushi.ripgrep.MSVC
winget install sharkdp.fd
winget install UniversalCtags.UniversalCtags

git clone <your-repo> vimrc
cd vimrc
.\install.bat
#+END_SRC

*** Linux/macOS

#+BEGIN_SRC bash
# Install tools
brew install ripgrep fd ctags  # macOS
sudo apt install ripgrep fd-find universal-ctags  # Ubuntu

# Clone and install
git clone <your-repo> vimrc
cd vimrc
chmod +x install.sh
./install.sh
#+END_SRC

* Configuration

** User Environment (~local/user_env.vim~)

#+BEGIN_SRC vim

g:mcge_custom_author = "Your Name"

* Troubleshooting

** CoC Not Working

1. Check Node.js: ~node --version~ (requires 16+)
2. View status: ~:CocInfo~
3. Restart CoC: ~:CocRestart~

** Search is Slow

1. Ensure ripgrep is installed: ~rg --version~
2. Use Clap: ~:Clap files~

** Module Loading Issues

#+BEGIN_SRC vim
:VimrcLoadReport         " Check failed modules
:echo g:mcge_startup_time " View startup time
#+END_SRC
