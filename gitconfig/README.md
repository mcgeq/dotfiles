# Git Configuration

Global `.gitconfig` with user info, aliases, and diff settings.

## File

- `gitconfig` — Global git configuration file

## Includes

- User name and email
- Line ending and encoding settings
- Git aliases (shortcuts)
- Delta diff viewer configuration (if configured)
- Credential helper

## Notes

- `hooks` and `commit.txt` are deprecated — use `commit/` (husky + commitlint).

See [gitconfig](gitconfig) for full configuration.
