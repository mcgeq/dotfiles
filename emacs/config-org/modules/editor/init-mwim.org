* MWIM Module

本模块配置 mwim (Move Where I Mean) 包，提供智能的行首/行尾移动功能。

** MWIM Configuration

MWIM 提供智能的行首/行尾移动。

#+begin_src emacs-lisp
;;; ============================================================
;;; MWIM Configuration
;;; ============================================================

(defun mcg-mwim-setup ()
  "Setup mwim package."
  ;; Bind C-a to mwim-beginning: toggles between code-beginning and line-beginning
  (mcg-bind-key "C-a" #'mwim-beginning "editor/mwim")
  
  ;; Bind C-e to mwim-end: toggles between code-end and line-end
  (mcg-bind-key "C-e" #'mwim-end "editor/mwim")
  
  ;; Optional: bind home/end keys
  (mcg-bind-key "<home>" #'mwim-beginning-of-line-or-code "editor/mwim")
  (mcg-bind-key "<end>" #'mwim-end-of-line-or-code "editor/mwim")
  
  (mcg-log "Configured mwim"))
#+end_src

** Initialize

#+begin_src emacs-lisp
;;; ============================================================
;;; Initialize
;;; ============================================================

(autoload 'mwim-beginning "mwim" nil t)
(autoload 'mwim-end "mwim" nil t)
(autoload 'mwim-beginning-of-line-or-code "mwim" nil t)
(autoload 'mwim-end-of-line-or-code "mwim" nil t)

(mcg-mwim-setup)
#+end_src
