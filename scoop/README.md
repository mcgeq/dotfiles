# Scoop Configuration

Windows package manager configuration for Scoop.

## File

- `scoop` — Scoop config directory

## Settings

- **Root path**: `D:\devTools\Scoop`
- **Global dir**: `D:\devTools\Scoop\GlobalScoopApps`
- **aria2**: Enabled (faster downloads)
- **Repo**: `https://github.com/ScoopInstaller/Scoop`
- **Branch**: `master`
